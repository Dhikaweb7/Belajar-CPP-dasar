## Apa itu C++ 


C++ adalah bahasa pemrograman komputer yang dibuat oleh [Bjarne Stroustrup](bs.md), 
yang merupakan perkembangan dari bahasa C yang dikembangkan di Bell Labs oleh [Dennis Ritchie](dr.md). 
Pada awal tahun 1970-an
